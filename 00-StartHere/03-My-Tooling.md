# 03 使用ツール

## ローカル環境チェック

- Salesforce CLI (`sf`)
- VS Code + Salesforce Extension Pack
- Git
- Node.js LTS（LWC / ツール実行用）
- 任意: Postman, jq, Mermaid

## 推奨コマンド

```bash
# 全文検索
rg "keyword"

# Markdown ファイル一覧
rg --files -g "*.md"

# 差分確認
git status --short
```

## 日次ワークフロー

1. 新情報は `99-Inbox/_to-triage.md` に記録。
2. 整理時に `01-Clouds` か `02-Shared` へ振り分け。
3. 案件背景が必要な内容は `04-Projects` に残し、知識側へリンク。
